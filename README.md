# Pokédex App

Esta es una aplicación web hecha con **React** donde se pueden ver diferentes Pokémon.
La app usa una API pública llamada **PokéAPI** para obtener la información.

## ¿Qué se puede hacer en la app?
- Ver una lista de Pokémon
- Entrar al detalle de cada Pokémon
- Ver su imagen, tipos y estadísticas
- Marcar Pokémon como favoritos
- Ver la lista de favoritos
- Navegar entre pantallas sin recargar la página

## Herramientas que usé
- React
- Vite
- React Router DOM
- Axios
- PokéAPI
- LocalStorage
